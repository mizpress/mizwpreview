Free real estate plugin for WordPress that lets you create, manage and list properties

== Description ==

Netune Real estate is a free WordPress plugin that lets you create and manage real estate properties. This plugin was built to work with the [free Neptune Real Estate Theme](https://thepixeltribe.com/template/neptune-real-estate/ "free wordpress real estate theme")


== Want More Features? ==
we've built a pro version of this plugin that let's you do a lot more:

 * Builder page with dynamic sections 
 * list properties by location,type (sale/rental)
 * Add and assign agents to the properties
 * Use google maps
 * Property Videos & Floor plans
 * List Agents/Teams
 * Search Property
 * One-On-One email support

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `neptune-real-estate.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
